UI - Containers

These are parts of the skeleton that hold menus and other widgets.

Common Bar

This holds large icons only - up to 9. Six are visible on tablets, with 4 being visible on 'phones.

File Port

This is the large area where files are opened. All programs can open more than one file, in tabs. Each tab includes the file name. This is just the file port and tabs, it doesn't include the file loading and saving logic, or the tab /title bar naming indicator to show the correct file names.

Menu Bar

The menu bar holds up to 9 menus, which drop down from the top. Only 4 are visible on 'phones, 6 on tablets, and 9 on PC's.

Status Bar

The status bar is below the file port and contains a settings menu for changing theme or opening other programs. This gets updated via plugins returning a value to update_status widgets.

Tool Pane

This appears on the right, and holds 1 or more tool boxes and any help messages or errors. On phones, this is disabled. The help box is written to via other plugins, information also appears here.
