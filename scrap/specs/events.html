Events

Listener

A process that listens for user input from a variety of keys, passed in as an array of keyboard shortcuts or function keys. When it catches a key event that matches one of the shortcuts, it pops up the associated dialog, for that menu item or action.

Keyboard Shortcuts

Input: One keyboard shortcut and the dialog, quick menu or plugin to open.
Output: An associative array mapping shortcut (keycode lists) to program calls.

System

This detects whether the browser tab has closed, and warns the user whether to save their documents or not. If a database connection is open, this closes the connection.

If connection loss occurs, the last changes are written to the file and placed in a recovery file, and encrypted. Encrypted files won't be deleted. The session is then closed.

On the sales site login, the user will be notified of any crashes, and the offer of recovering the data will be made.
