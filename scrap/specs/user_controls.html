UI - User Controls

These user controls get copied into the quick menus and dialogs as well as the main app itself.

Button

A button which can be a submit button or a button that toggles the display for half of a form, dialog, quick menu.

Checkbox_wLabel

A checkbox with the associated label on the left. Can be placed in a fieldset group to separate out blocks of controls.

Clipboard Entries List

An 'entries that have been copied' list with small text only previews on the right and radio buttons on the left or just a combo box with scroll. At the bottom are two buttons, paste and clear. A quick menu.

Colour Picker

A full colour wheel that picks one colour and displays the technical details for that colour. A quick menu.

Colour Swatches

A set of colours in groups. A combo box on top selects the colours displayed. The user can add their own palettes. A quick menu.

Combo Box

Like a select list but with no drop down element. Can have a custom height. The width can also be changed. Can also contain pictures instead of text for each option in the list.

Currency Field

A currency field is a text input that only accepts currency style decimal numbers. The currency sign depends on the locale setting and doesn't get stored.

Date Field

This stores the date in a format determined by the locale which is either European (1-1-14), UK (01/01/2014) or US (mm-01-2014). A little calendar date picker is also used for a selection alternative.

Email Field

This checks for a valid email, with valid domain name ending.

Float Field

This allows 4 number precision, and converts whole numbers entered into floats.

Icon

A small clickable image to represent one single action or function of a program.

Large Icons

A icon with text to the right, for commonly used features that should be easier to prod and find. Text up to 10 characters long.

Long Text Field

A text area field with no rich edit capabilities except in email or project management type programs. Contains a spellchecker.

Progress Bar

A small bar to describe how long something is taking. Horizontal with a percentage at the end.

Radio Btns wLabel

A radio button with a label on the right. Belong in a group. The name of this control will apply to all buttons, with each buttons value being that of the label.

Sci_num_field

This an exoponent number field for high precision numbers. Save the exponent part separate as well as the mantissa and decimal parts. Example: 201.23416789(e-10) save exp: int exp = -10; Saving mantissa: int mantissa = 201; Saving decimal: int (long) dec = 23416789;
Save the whole number as an array e.g. number[mantissa,dec,exp].

Spin Box

A small box with either a whole or float number that can go up or down. The user specfies by what step, and the lower and higher thresholds.

Text Field

A small, fixed width text input.

Time Field

A field just for the time. A small clock to pick the time could also be used. Displays in a locale specific format and also saves the users configured time zone.

Tool Tip

An extended yet 80px high tool tip for offering information and help. About a sentence of text. The keyboard shortcut of something should always come first in this sentence.

Update_Status_Msg

This creates a variable for the status bar with a label. When this control is updated, that underlying variable also changes. Set to update every minute by default.

Upload Pic

This offers a text label area and an upload button with a picture icon. The text label area displays 'no picture uploaded' when there is no picture. If the picture is not the correct format, it results in an error. This returns the picture URI, any file name has spaces replaced with underscores. A quick menu.

Upload Text

This uploads a text file in which to insert after the current cursor position in the current file. If a program is uploaded, the executable permissions are stripped, then the special characters are translated into character entities. Similiar visually to upload pic.

URL Field

This lets the user input a web address. It adds the http:// automatically and checks the domain format.

Whole Num Field

This lets the user input only a whole number. If a float is inputted, the output is truncated and the user is warned. The user is warned if they try and enter anything else.

Custom Field

This lets a user select a regex for, say phone numbers or initials. The pattern and character limit as well as the label make up the variables for that new field, which goes in their application somewhere.
