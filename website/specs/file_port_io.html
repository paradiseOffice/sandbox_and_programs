File IO

New File

This adds a new file tab into the file port which gets populated via the inner PHP require_once.

Open File

This dialog opens a file by uploading the file to the server.

  1. The file is checked for viruses and Windows nasties.
  2. The file gets moved to the secure upload folder and spaces replaced with underscores in the name.
  3. The Python conversion is ran on it if it's not odf.
  4. Otherwise, an XSLT transform is ran on it, turning it to HTML. The CSS styles are appended to the <style> block in the main app.
  5. The HTML is amended into the file port (app specific).

Save, Save As, Save All

  1. The CSS for file x selectors only is written into a simple HTML file header.
  2. The HTML in the file port gets added to the body of the above file.
  3. The XHTML gets converted back into odf.
  4. This is either downloaded to the user as an attachment.
  5. Or it gets converted into a binary file format.
  6. That document gets downloaded.

Print

This opens up the OS print dialog, and sends the document with print.css applied. If the user selects print, enters the number of copies etc, the document gets printed.
