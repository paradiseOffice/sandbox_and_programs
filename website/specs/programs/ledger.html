Data format - accounts list. CSV. Use Sqlite3 zipped up.

File Menu

New - toggle

Input: Replaces accounts already open.
Output: If there are accounts open, these get saved and replaced with a new set of accounts.

Accounts List - Quick Menu

Input: User selects a CSV list of accounts or can create a new list.
Output: A list of accounts to do double entry transactions with.

Open File - Quick Menu

Input: The user selects a CSV file.
Output: The CSV file gets imported into the current account.

Save - Quick Menu

Input: The user enters a file name, and the only file format available is sql.
Output: A zipped sql file.

Save As - Quick Menu

Input: Same as save but with a differing filename.
Output: A zip file with a different filename.

Print View - Quick Menu

Input: A ledger or report/chart that is currently selected.
Output: The print view for the item currently selected.

Print - Quick Menu

Input: The current ledger or report.
Output: A print view, and print out.

Close - toggle

Input: The currently selected ledger or report.
Output: Closes the report, or ledgers tab.

Edit Menu

Undo - toggle

Input: A click
Output: Undos the last entered character or formatting command.

Redo - toggle

Input: A click
Output: Redos the last entered character or formatting command.

Cut - toggle

Input: A selection
Output: Saves the selection to clipboard, and removes the selection.

Copy - toggle

Input: A selection
Output: Saves the selection to clipboard.

Paste - toggle

Input: A cut or copied selection.
Output: Pastes the numbers or text back in.

Paste Special - Quick Menu

Input: A HTML or text selection
Output: Either keep the HTML formatting or strip the tags.

Go To - Quick Menu

Input: A transaction number and ledger.
Output: Go to that transaction number in a ledger.

Accounts - Dialog

Input: The user can enter different accounts.
Output: Each account also has categories and other details like addresses.

Find &#amp; Replace - Quick Menu

Input: User enters text or numbers to find, and a possible replacement.
Output: By default the user has to OK each replacement, and finds are highlighted, and any row numbers highlighted.

View Menu

Zoom - Quick Menu

Input: Can zoom in or out, or use a slider.
Output: The tables are zoomed in or out.

Filter - Quick Menu

Input: The user selects an account to filter by.
Output: Only transactions to a certain account are shown in the table.

Ledgers Menu

Purchase Ledger - Tab

Input: Transactions for large fixed price purchases that aren't day to day expenses.
Output: A separate database table for these fixed transactions, with a date and hire purchase details.

Invoice Ledger - Tab

Input: Invoice based transactions.
Output: A database of invoice based transactions with an invoice 'report' HTML,PDF file.

Sales Ledger - Tab

Input: Every sale made for products.
Output: A transaction table specifically for product based sales including a product code and payment method.

Income - Tab

Input: An aggregate of all sales, and any service transactions that generate income.
Output: A table specifically for service based income, and product sales.

Expenditure - Tab

Input: Operational expenses that happen on a day to day basis, can be wages, stationery or electric bill kind of things.
Output: A table of all expenditure transactions with a frequency column.

Tax returns - Dialog

Input: The profit/loss and other things needed to work out tax. The user enters their country (only UK for now).
Output: A tax return calculator that can also update HMRC or something like that.

VAT

Input: Optional if the company is big enough to need VAT.
Output: Works out the VAT owed or whatever.

Payroll - Tab

Input: The employees, their salaries and time working in a week.
Output: Option to generate time sheets. Also manages NI payments and employee benefits such as child care. A separate set of tables.

Borrowing - Dialog

Input: Credit cards, loans or credit agreements.
Output: Details of any repayments, along with an interest calculator and warnings for late payments, or underpayments of debts.

Cash Flow - Tab

Input: The income, sales, and expenses, purchase ledgers over the course of a month.
Output: A graph showing the income flows with optional detail.

Forecasts - Dialog

Input: Number estimates for income and expenditure.
Output: A forecast for a month up to 3 years for a business - like a large cash flow statement.

Insert Menu

Formula - Quick Menu

Input: Select a formula group, and then a formula with an edit bar.
Output: The formula calculates on selected table columns, doing the calculation for each row.

Conversion Field - Quick Menu

Input: A starting currency and a currency to convert to.
Output: By default converts from GBP to dollars. Uses conversion information updated by the day.

Chart - Dialog

Input: A range of transaction rows from a selected ledger.
Output: A chart in whichever style the user chooses.

Goalseek - Quick Menu

Input: A desired value of sales or whatever to achieve.
Output: A report of various ways of seeking that goal and how long it will take, under different circumstances.

What If Analysis - Dialog

Input: Different variables to measure such as staffing levels or investment in plant, machinery etc.
Output: Different charts and data of how that would affect the business (generates reports).

Headings - Quick Menu

Input: Change the default headings for a currently used ledger.
Output: Renamed table columns, or table column aliases.

Report - Dialog

Input: The user selects a group of transactions in any ledger, with the option of printing different ledger transactions in a split view and any optional charts.
Output: The user can also select a lot of formatting options or some available presets.

File - Quick Menu

Input: Import a CSV file, into the currently selected ledger.
Output: Has the option of specifying which fields go into which table columns.

Format Menu

Ledgers - Quick Menu

Input: An open set of accounts.
Output: Changes the formatting presets for the table views for each ledger. Complete with accessible presets.

Heading - Quick Menu

Input: A currently selected ledger or report.
Output: Change the look of the heading for that ledger only. Overrides the preset themes.

Numbers - Quick Menu

Input: Changes inputted numbers into specified formats.
Output: This is across all the ledgers. Reports can have different number formatting.

Chart - Dialog

Input: A currently open chart.
Output: Format each part of a chart or change the chart type and alter the legend.

Report - Dialog

Input: A currently open report.
Output: Formats various parts of the report, but not any charts in them.

Account - Quick Menu

Input: Change preset theme for a selected account.
Output: Choose between the ones available. The heading still has the same formatting as set earlier.

Page - Quick Menu

Input: Enter print settings.
Output: Pages print on landscape by default.

Data Menu

Sort Column - Quick Menu

Input: User selects a column
Output: Can sort A-Z or Z-A, ascending or descending

Create Formula - Dialog

Input: A mathematical formula and some column references.
Output: A calculated column.

Currency - Quick Menu

Input: Select a currency to use by default.
Output: Includes formatting options.

Validation - Dialog

Input: Users can build their own input patterns to be used inside that ledger.
Output: A validated set of data.

Options

Company Details - Quick Menu

Input: Company name, organisational structure
Output: Also the legal framework, e.g. PLC, Ltd, sole trader which affects the accounts.

Change Language - Quick Menu

Input: A language
Output: The accounts change into that language, the currency being separate.
