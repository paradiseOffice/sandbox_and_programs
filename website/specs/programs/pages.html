Pages

Tag Line

Type up text rich documents or change existing files.

App Description

Change, print and format documents by styling text, including simple tables and images or create your own document from scratch. Opens common file types from Libreoffice and Microsoft Word. Also can be used to edit and save PDF files.

Pages+ description

As with pages, without any advertisements. More data source options for database or addressbook intergration. Mail merge, the ability to record and play back a sequence of actions and a simple footnotes utility. Access and modify document information, or check the grammar of text. More file saving customisation and a simple biblography.

Features

Use Template - Dialog

Input: Several templates on web server (already in HTML, converted from ODT/DOC).
Output: User selects tab with document type (e.g. letters), then the template preview. The template is copied into a new tab ready to be edited.

Save All

Works in similiar way to Jotter, saves up to 5 open documents, one at a time from the 1st tab to the last.

Document Info (+)

Input: An open file or new file.
Output: Displays the organisation, name, creation and modification date and any editing statistics for the file.

XML Settings (+)

Input: An open document, a user specified XSL file.
Output: Upload an XSLT stylesheet to parse the XHTML with on save.

Print View - Large Quick Menu

Similiar to Jotter, but with formatting. Shows greyscale if that option is selected.

Print

Prints, then exits. Pops up a print preview dialog quickly to format the HTML correctly.


Undo

Similiar to Jotter

Redo

Similiar to Jotter

Undo History - Quick Menu

Input: 1 - 20 words entered or formats. A format through a dialog counts as one group of formatting.
Output: List of formats, words changed, from newest to oldest.

Paste Special

Input: A copied snippet of HTML.
Output: Plain text, all formatting and tags stripped off.

Clippings - Quick Menu

Input: Copied or cut text.
Output: Show formatting of texts. 20 clippings, 6 shown initially and scroll down for more.

Select All

Selects all of the current file including pictures, tables and other things.

Styles - Quick Menu

Input: Selected text. User can also create styles.
Output: Brings up a list of formatted styles, making it easy for the user to select one. Styles are organised in groups, with the two main headings being character and paragraph. Then user styles, system styles, default styles.

Colours - Quick Menu

Input: The user selects colours from a colour picker or a palette to use.
Output: Those colours are available in colour swatches elsewhere in the program.

Find & Replace - Quick Menu

Input: Find a word, and enter a word to replace it if needed.
Output: Can choose how many pages, documents or selection(s) to look through. Does both forward and back search by default. Can search just headings. Replaces 1st word only and confirms (default). Replace all highlights replaced words, changing them, then asks for confirmation, with the dialog moved into the right-bottom corner.


Zoom - Quick Menu

Input: Current document, along with the current zoom setting.
Output: Can zoom in or out using a slider or spinboxes. 2 or 4 pages in one screen can also be displayed, or 3 pages going across.

Page Marks - Quick Menu

Input: A selection in a document -> turn into Page Mark by clicking 'add'.
Output: A list of places within the document like pictures and headings, or these bookmarks to jump to.

Header & Footer - Dialog

Input: Size, fields in header and footer.
Output: Adds header/footer to the pages, makes area for text slightly smaller. The fields can be deleted, and the areas (always visible once enabled) can be written into.

Changes - Toggle

Input: Menu item shows tick after being clicked.
Output: The changes the logged in user makes are marked in blue, in the margins. Those changes are saved in the document.

Data Sources (+) - Quick Menu

Input: User selects a spreadsheet or CSV file.
Output: The fields are shown, and each field can be named for use as a data field.

Data Fields (+) - Quick Menu

Input: A data source can be loaded. The (named) fields from user data sources can be selected, as can date/time, name, page numbers etc.
Output: The field appears after the current cursor position in a little box with a dotted black border.

Layout - Quick Menu

Input: Select from 2 pages, outline, print and web page.
Output: The page view changes accordingly.

Set Tabs - Dialog

Input: The user selects the kind of tab, and how many with the aid of a ruler (and visual options for leader lines).
Output: Tabs are displayed in the current document, as little tabs near the top of the page, with a blue line when clicked on. Colour coded and static. Only alterable through the dialog.

Non-printing Characters - Toggle

Input: Click for on/off. Little tick to show it's on.
Output: Shows dot for each space, line ends etc, on light lemon yellow background for the characters.


Insert Menu

Page Break

Input: Current file and cursor position.
Output: Breaks the page or displays black line in default web page view.

Character - Dialog

Input: Select a font and glyph set, choose a character.
Output: Table of 4 rows x 20 columns - for glyphs. User picks one and the glyph gets inserted into the document via unicode or XML (whichever ODT currently uses).

Landscape Page

Input: Current file in portrait orientation and cursor position.
Output: Adds a landscape page after a page break, then another page break. Other pages are portrait.

Columns - Quick Menu

Input: Current file. Pick 1-3 columns, input space in between columns and how many pages have columns as a page range.
Output: Columns for the specified amount of pages. Shown in light grey (fading to transparent?).

Table - Dialog

Input: Pick how many rows/columns in table, and whether it has a title(thead) and width/height.
Output: Places the table in the current file after the cursor position.

Text Box - Dialog

Input: Asks for width/height and rounded corners / shadow.
Output: Adds to current file after cursor position.

Comment - Quick menu

Input: The user types in a comment and selects the colour.
Output: The last word before the cursor is the comment anchor and is underlined / boxed in yellow. On hover, the comment appears in a post-it note style yellow box. The colour of the note can be changed.

Field (+) - Quick Menu

Input: The user selects and adds a field, which can be read-only.
Output: A grey area for the documents reader to fill in, comes after the cursor, again.

Link - Quick menu

Input: The user either selects a web link, or types it in.
Output: A traditional style hyperlink that opens in a new tab when clicked on. Pointer changes to a little hand on mouse over.

Picture

Input: The user uploads a picture, or enters a picture URL.
Output: The selected picture is uploaded or fetched, filename sanitised, and linked into the document. A border, caption or simple transform can also be done on the picture. Gets added after the cursor.

Movie & Sound - Dialog

Input: Adds a movie or sound - and tells the user it will only work in certain situations.
Output: A link to a video, or an embedded movie/sound clip with a caption (and explanations if it doesn't work).

File

Input: The user selects a doc/html/odt/txt file to insert.
Output: The file is uploaded, sanitised and added after the current files cursor position.

References (+) - Quick Menu

Input: The references are added and a link created.
Output: The selected text gets the link, the reference goes in the footer of each page by default.


Format Menu

Clear Formatting

Input: A selection.
Output: Clears all formatting, and tags - leaving plain text.

Change Case - Quick Menu

Input: A selection of text.
Output: Either title case, upper or lower case.

Character - Dialog

Input: Grouped by readability, then effects like colour/underline etc. A selection of text, a style name is optional.
Output: The chosen font etc makes a new character style, say 'char_user1' (or the inputted style name) and is then applied to the text.

Paragraphs - Dialog

Input: Grouped by alignment, text, and pretty effects.
Output: The selections and choices applied to the selected paragraphs, and saved as a style, either 'para_user1' or the style name.

Lists - Dialog

Input: Grouped by bullets, pics, numbers. The user selects some text, and picks a list setting.
Output: The list is created, and formatted correctly.

Section - Quick Menu

Input: User enters a name for the section. The current active page where the cursor last was is the section start. User can alter that and the section end.
Output: A named section that automatically has a page mark set to the start. A section is created automatically with multicolumn layouts as well.

Page - Dialog

Input: Grouped by page setup and background. Page orientation, margins by section. Different background options.
Output: Every page or every page in that (named) section having those settings.

Table - Dialog

Input: Group by sizes, prettiness, text wrap. Select a table then set the formatting options.
Output: The table is then reformatted correctly, or new rows / columns added.

Text Box - Dialog

Input: Group by size, prettiness, text wrap. Select a current text box - and choose formatting options.
Output:A correctly formatted text box.

Picture - Dialog

Input: A selected picture. Cropping or colour alterations.
Output: Either a cropped or greyscale picture, or one with a border.

Biblography (+) - Quick Menu

Input: References set earlier. References on bottom of page, or collect together in biblography by page number/line no.
Output: Either footnotes or a biblography.


Table Menu

Insert - Dialog

Same dialog as Insert-Table

Rename - Quick Menu

Input: A selected table, and a table name
Output: Can find the table by name.

Head - Dialog

Input: Creates headings for the Columns, with formatting options.
Output: Nice colourful bold headings that are fixed there by default.

Foot - Dialog

Input: Creates footers for the Columns, with formatting options.
Output: Nice bold footer that aren't fixed there by default.

Merge/Split Cells - Quick Menu

Input: A few selected table cells, option to merge or split.
Output: Either merged cells, or split cells if acceptable.

Table Designs - Dialog

Input: A selected table, or named table from drop-down list.
Output: The user selects a nice table design, and the table is formatted that way.

Alignment - Quick Menu

Input: A selected table
Output: Either left, center or right aligned within the page or in the centre of one page.

Text Wrap - Quick Menu

Input: A selected table
Output: Wrapping the text after the table, or tightly round the table.

Properties - Dialog

Input: Can change the selected tables names, and the width and height.
Output: Changes the table names, and the width and height. Can also change row and column heights.


Add Language - Quick Menu

Input: Select a language to use for that file.
Output: The file language changes, and the spell check uses that locales dictionary.

Change Language - Quick Menu

Input: Select one or two languages to use. Selected text can be marked as a particular language.
Output: Can have sections in two different languages.

Spellcheck - Quick Menu

Input: Goes through each word in the file, and runs a dictionary based spellcheck.
Output: Gives suggestions and a red underline to misspelt words.

Grammar Check (+) - Quick Menu

Input: The current file or selection
Output: A green underline, and English grammar rules with options. Not as finicky as Word.

Word Count - Quick Menu

Input: The current file, or selection.
Output: Counts the words, characters and heading word count.

Thesaurus - Dialog

Input: A selected word, a user specified thesaurus site.
Output: Related words are displayed, with definitions and opposite meaning words.

Encyclopedia - Dialog

Input: The user enters a term, and research sites like Wiki or Google.
Output: Text only research selections appear as links, pick to display more. Diagrams can be turned on or references followed. Links can also be opened in new tabs.

Outline Options - Dialog

Input: The current file with heading levels 1+
Output: Displays the differing number/bullet styles for the headings and lists.

Macros (+) - Dialog

Input: Some repitive tasks to be done. No record/stop buttons.
Output: A coded Javascript snippet that replays the word processor events.

Line Numbering - Toggle

Input: The current file.
Output: Numbers each paragraph and line like in a legal document or something.
