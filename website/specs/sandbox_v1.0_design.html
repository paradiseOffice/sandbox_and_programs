Detailed System Design

Cygnet Sandbox - App Creator

Definition:
Reads user choices in a multi-page form and outputs HTML, CSS, PHP, Javascript and Python files. Only works with current modules and parts available.

Responsibilities

  • To put the HTML and other code snippets in the right order.
  • To write standards compliant code that works across many browsers.
  • To encode SQL queries that work in any of the common databases, and that doesn't expose security risks.
  • To take in any options and validate them.
  • To keep the appearance and structural CSS apart at all times.
  • To provide consistent methods of linking programs to options.
  • To provide specific methods of data validation for office and web applications, for example to prevent 'divide by zero' in maths.
  • To provide default JQuery animations for the modern web.

Constraints

There will only be a few modules available initially, growing in number as time goes on.
